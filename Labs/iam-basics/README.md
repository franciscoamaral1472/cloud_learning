# IAM Basics Lab

## Objective
Understand how to manage users, groups, and permissions in AWS.

## Steps
1. Access AWS IAM service
2. Create a new user
3. Assign permissions (e.g., AmazonS3ReadOnlyAccess)
4. Create a group and attach policies
5. Add user to the group

## Key Concepts
- Users vs Groups vs Roles
- Policies and permissions
- Principle of least privilege

## Result
Successfully created and managed IAM users and permissions.
